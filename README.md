CampusMetric 是一套面向学校本地部署的开源校园量化评价系统。它通过考评字典动态生成教师端和管理端表单，支持多方案、材料审核、自动积分、长期档案和可选的 AI 材料识别。

> 开源仓库不提供公开下载；请按下文从源码自行部署运行。

## 技术路线

- Python 3.11+
- Flask + Flask-Login
- SQLAlchemy + SQLite
- Jinja2 + 本地静态资源
- Waitress 生产服务器

技术路线参考本地 OpenMentor 项目的轻量部署方式，系统数据保存在本机，不依赖 SaaS 平台。

## 当前已完成

- 首次运行创建超级管理员和当前学年。
- 超级管理员统一管理部门管理员、审核人、教师账号和全校 AI 配置。
- 所有角色均可在“我的账号”中验证当前密码后自行修改密码。
- 部门管理员可创建、切换、归档和恢复多套评价方案。
- 管理员登录、退出和基础工作台。
- JSON 空白模板、完整示例和字段说明下载。
- JSON 字典校验、预览、整套导入和当前字典导出。
- 考评字典中每个指标可展开查看具体计算方式和原始规则，并可直接进入手动编辑。
- 9 种首版积分类型的基础校验与计算。
- 教师端自动读取`teacher`来源指标。
- 管理员端自动读取`admin`来源指标。
- 根据积分类型动态生成录入控件。
- 管理员录入后直接计分；教师录入后进入待审核状态。
- 参考现代教育工作台的全站响应式 UI，桌面端使用侧边导航，手机端使用底部导航。
- 管理端移动访问中心，可为管理员录入、一级审核、教师填报和教师查看生成二维码。
- 管理员、审核人和教师账号的创建、启停与密码重置。
- 超级管理员下载 Excel 模板并批量导入账号，整表校验通过后原子写入。
- 教师填报、一级审核、退回原因和本人分数查看。
- 审核页按中文字段展示教师填报内容，证明图片在当前页面浮窗预览。
- 管理员查看全部考评记录、直接调整最终分数并保留操作日志。
- 考评方案编号与账号、学年、考评记录关联，为后续多方案预留。
- 字典 JSON 导入和手动配置并存。
- 图片上传自动压缩、材料预览和 OpenAI 兼容视觉模型证书识别；教师可先识别并按当前指标自动回填表单，再核对提交。
- 管理端总分排名、Excel 导入/导出、学年归档/还原和新学年字典继承。
- 安全的在线更新检查、SHA256 校验、暂存、代码备份与免安装包托管重启框架。
- PDF / Markdown 评价方案在系统内调用 AI 生成字典、校验并直接写入。
- 材料类指标逐条录入，每条只上传一张自动压缩图片。
- 附件按实际文件内容校验，仅接受 JPG、PNG、WEBP、HEIC/HEIF 图片。
- 跨当前与已归档学年的教师/班级档案趋势、同期对比和指标构成。
- 档案跟踪可跨当前账号有权限的全部方案和全部学年，并通过浮窗查看记录明细。

## 启动

macOS 或 Linux：

```bash
./start.sh
```

Windows 源码运行：双击 `start.bat`。

Windows 免安装版：解压后双击 `CampusMetric-启动.bat`，无需预装 Python。首次启动会自动生成本机 `.env` 和随机会话密钥。

首次启动会自动创建虚拟环境、安装依赖并生成`.env`。默认访问地址：

```text
http://127.0.0.1:5012
```

全新安装首次打开时需要自行创建超级管理员账号，不使用固定默认密码。旧版升级会保留原管理员为部门管理员，并创建本地迁移用超级管理员；请登录后立即重置密码。

## 测试

激活虚拟环境后运行：

```bash
python -m unittest discover -s tests -v
```

## 数据目录

- SQLite：`instance/evaluation.sqlite`
- 上传材料：`uploads/`
- 运行配置：`.env`

这些运行数据默认不纳入版本管理。正式使用后应定期同时备份数据库和上传材料目录。

## 安全提示

- 正式使用前请创建自己的管理员账号，不要保留公开测试密码。
- API Key 只保存在部署机器的 `.env` 或数据库中，不应提交到 Git。
- 在线更新会校验 SHA256，并在覆盖程序前备份数据库和被替换文件。
