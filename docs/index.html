让每一套校园评价方案，都能快速落地。

WHAT IT DOES

一套系统，承接多种考评事宜。

超级管理员统一维护账号与 AI，部门管理员各自管理多套方案；老师只看到自己所属方案和自己的分数。

◇

字典驱动

后台手动配置，或导入标准 JSON。指标、计分类型、分值、材料要求与班级跟踪字段全部可调整。

✦

AI 方案转换

直接上传 PDF / Markdown 评价方案，系统调用统一大模型生成规范字典、自动校验并写入。

▦

手机随时填报

管理员录入、教师填报、一级审核和分数查看均为手机响应式，并可生成模块访问二维码。

✓

单条材料留痕

一条评价记录对应一张压缩图片，证书可由 AI 识别；逐条录入保证次数与长期统计准确。

≡

多方案与归档

管理员可以新增、切换和归档多套方案。归档后只读，需要明确还原后才能修改。

⌁

长期档案趋势

按教师或班级跨学年查看累计得分、月度趋势、同期对比与指标构成，已归档数据也参与分析。

ONE SIMPLE FLOW

从纸面方案到可运行系统。

01 / CONFIG

创建方案

为不同部门和考评事宜建立独立方案编号与学年。

02 / DICTIONARY

配置字典

手动配置、JSON 导入，或让 AI 直接读取原方案。

03 / COLLECT

录入审核

教师逐条提交材料，审核人一级审核；管理员数据直接生效。

04 / ARCHIVE

汇总沉淀

得到总分与排名，学年结束后归档，并持续形成长期档案。
