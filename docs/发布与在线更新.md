# 发布与在线更新

## 更新源格式

当前公开更新源：`http://121.199.56.216/campus-evaluation/updates/`。

更新源目录包含：

- `version.json`
- `campus-evaluation-update-x.y.z.zip`

`version.json` 字段见 `release/version.example.json`。系统会校验 SHA256，解压时防止路径穿越，更新时显式排除：

- `instance/`（SQLite）
- `uploads/`（考评材料）
- `backups/`
- `.env`
- `.venv/` 和免安装包的 `env/`
- `.git/`

应用更新时除备份被替换的代码外，也会先备份 `instance/evaluation.sqlite` 到同一回滚目录；数据库兼容迁移只在新版本重新启动后执行。

## 生成更新包

```bash
python scripts/build_update_package.py 0.3.0
```

输出到 `dist/`，包含 zip 和已计算 SHA256 的 `version.json`。

## 免安装包

Windows 免安装包根目录包含完整项目代码和预装依赖的 `env/` Python 运行时，用户双击 `CampusMetric-启动.bat` 即可运行。启动器会设置 `CE_MANAGED_LAUNCHER=1`，在线更新应用后可自动重启。

公开下载地址：`http://121.199.56.216/campus-evaluation/downloads/CampusMetric-Windows-Portable-0.3.0.zip`。当前构建脚本使用 Python 3.13.14 官方 Windows x64 embeddable runtime，并下载与之匹配的 Windows wheels；发布前必须检查 ZIP 完整性、依赖文件和敏感文件排除结果。

源码开发模式可检查、下载、校验和暂存更新，但不会自动覆盖当前开发目录。

为确保旧版免安装包在无网络环境升级后也能读取 PDF，更新包内置纯 Python `pypdf` 运行库及其 BSD-3-Clause 许可证；正常源码安装仍以 `requirements.txt` 为准。

## 开源发布边界

公开仓库和发布包不得包含 SQLite 数据库、上传材料、AI API Key、学校正式账号或私有更新源地址。后续固化并推送 GitHub 时应使用 `buBailai` 身份。
